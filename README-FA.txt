# LAN Queue v12 (Railway-ready)
- Node.js بدون وابستگی خارجی
- ورود/خروج، مدیریت کاربران (ادمین)، نوبت‌دهی، ثبت عکس، ثبت چاپ، گزارش آمار، خروجی XML/CSV، فراخوان صوتی فارسی
- آمادهٔ استقرار روی Railway (پورت از `process.env.PORT`)

## استقرار روی Railway (رایگان)
1) این پوشه را در یک ریپوی GitHub قرار دهید.
2) در railway.app وارد شوید → New Project → Deploy from GitHub → ریپو را انتخاب کنید.
3) بعد از Build، لینک HTTPS دریافت می‌کنید.

> توجه: فضای فایل در پلن رایگان دائمی نیست. برای ذخیره پایدار از PostgreSQL رایگان Railway استفاده کنید.
